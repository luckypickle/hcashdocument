◆ relay_account_name

string hsrcore::client::Config::relay_account_name

relay account name is used to specify the name of the account that must be paid when network_broadcast_transaction is called by a light weight client. If it is unset then no fee will be charged. The fee charged by the light server will be the fee charged light_relay_fee for allowing general network transactions to propagate. In effect, light clients pay 2x the fees, one to the relay_account_name and one to the network delegates.
