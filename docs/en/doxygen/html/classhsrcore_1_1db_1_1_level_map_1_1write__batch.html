Detailed Description

template<typename Key, typename Value>
class hsrcore::db::LevelMap< Key, Value >::write_batch

this class allows batched, atomic database writes. usage: { write_batch batch = _db.create_batch(); batch.store(key1, value1); batch.store(key2, value2); } when the batch goes out of scope, the operations are commited to the database
